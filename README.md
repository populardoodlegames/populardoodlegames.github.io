# Popular Doodle Games

Welcome to **Popular Doodle Games**!  
This website is a collection of fun and popular online games, designed for easy access and enjoyable gameplay.

## Features

- Play a variety of popular games directly in your browser.
- No downloads or installations required.
- Responsive design for desktop and mobile devices.
- Regularly updated with new games and features.

## Folder Structure

- `assets/css/` - Stylesheets for the website.
- `assets/img/` - Game images, icons, and banners.

## How to Use

1. Open the website in your browser.
2. Browse the game list and select your favorite game.
3. Click to play instantly!
